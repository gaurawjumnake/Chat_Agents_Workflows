# bug_analysis.snippet.md

## Use
Convert failures into reproducible bug reports and regression tasks.

## Prompt
```text
Analyze this defect.
Input:
- Symptom: <short>
- Reproduction steps: <steps>
- Logs/traces: <trimmed>
Output:
1) Defect summary
2) Probable root cause area
3) Reproduction confidence
4) Regression test recommendation
```
