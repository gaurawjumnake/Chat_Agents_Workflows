# edge_case_generation.snippet.md

## Use
Generate edge and abuse cases from spec and risk areas.

## Prompt
```text
Generate edge and abuse cases.
Input:
- Spec path: <path>
- Risk hotspots: <bullets>
Output:
1) Boundary cases
2) Invalid input cases
3) Timeout/retry/concurrency cases
4) Expected assertions
Keep under 250 tokens.
```
