### Windsurf

**Type:** IDE Agent (VS Code fork, owned by Cognition since 2025)  
**Model:** SWE-1.5 (proprietary, 950 tok/s via Cerebras), Claude, GPT  
**Pricing:** Free (limited) · Pro $15/mo · Teams $35/user/mo  
**Repo/Docs:** https://docs.codeium.com/windsurf/getting-started

Best price-to-feature ratio among AI-native IDEs. Cascade system automatically indexes large codebases (500+ files) without re-prompting. Proprietary SWE-1.5 model runs at 950 tokens/second via Cerebras — the fastest inference speed of any tool here. Acquired by Cognition (makers of Devin) in 2025, who also developed its government offering.

**When to use:**
- You want Cursor-level features at a lower price ($15/mo vs $20/mo)
- You need the fastest token generation speed
- Cost predictability matters — flat monthly pricing, no credits

**When NOT to use:**
- You need JetBrains support — VS Code fork only
- You need best-in-class reasoning depth — use Claude Code for that

**Accuracy:** High. SWE-1.5 is competitive. Multi-model access for complex tasks.

**Latency:** Very fast — fastest inference in this list (950 tok/s on SWE-1.5).

**Token efficiency:** High. Cascade auto-indexing reduces redundant file reads.

**IDE support:** VS Code-based (standalone)

**MCP:** Yes

**Privacy:** Cloud. FedRAMP High authorization for Windsurf's government offering.

**Limitations:**
- VS Code only
- Owned by Cognition — long-term roadmap uncertainty
- SWE-1.5 is capable but not as strong as Claude Opus for complex reasoning
